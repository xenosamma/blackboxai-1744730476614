# Precious Waste Refinery Website

## Project Overview
The Precious Waste Refinery website serves as a platform to connect with audiences interested in responsible electronic waste recycling. It promotes sustainable electronic waste management through education, engagement, and actionable steps for recycling e-waste. The website is designed to be user-friendly, modern, and accessible across all devices.

## Installation
To get started with the Precious Waste Refinery website, follow these steps:

1. **Clone the repository** to your local machine:
   ```bash
   git clone https://github.com/yourusername/precious-waste-refinery.git
   ```

2. **Navigate into the project directory**:
   ```bash
   cd precious-waste-refinery
   ```

3. Open the `index.html` file in your web browser to view the website.

## Usage
The website has various sections which include:
- **Header**: Navigate through the website and call-to-action buttons.
- **Hero Section**: Introduction and call-to-action for joining the recycling initiative.
- **Our Impact Section**: Showcases statistics related to e-waste recycling.
- **Services Section**: Details the types of e-waste accepted.
- **How It Works Section**: Explains the recycling process.
- **Join the Movement Section**: Encourages participation in recycling efforts.
- **Contact Section**: Contact form and additional information.

Navigate through the sections using the menu in the header to access different parts of the website.

## Features
- Modern and clean design with a focus on sustainability.
- Responsive layout that works well on both desktop and mobile devices.
- Interactive elements, including animated statistics and smooth scrolling.
- Informative sections that educate users on e-waste recycling.
- User-friendly navigation with clear call-to-action buttons.

## Dependencies
The project uses the following external libraries:
- [Tailwind CSS](https://tailwindcss.com/): Utility-first CSS framework for styling.
- [Font Awesome](https://fontawesome.com/): Icons for various sections.
- [Google Fonts](https://fonts.google.com/): Poppins and Inter fonts for text styling.
